# 校园二手交易平台（半成品）

一个基于 Vue 3 + Element Plus 开发的二手交易平台，提供便捷的二手商品交易服务。

- �� 商品搜索：支持关键词搜索、分类筛选、价格排序

- 💰 交易系统：在线购买、订单管理
- 📱 响应式设计：完美支持移动端和桌面端

## 技术栈

- 前端框架：Vue 3
- UI 组件库：Element Plus
- 状态管理：Pinia
- 路由管理：Vue Router
- HTTP 客户端：Axios
- 工具库：Day.js
